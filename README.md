# Spotify Clone (Vanilla HTML, CSS & JavaScript)

A simple front-end recreation of core Spotify UI/UX elements using only vanilla web technologies.  
This project focuses on layout, styling, responsive design, and basic interactivity such as play/pause controls.

## Features
- Responsive Spotify-style layout  
- Play/pause functionality  
- Song list UI  
- Progress bar simulation  
- Minimal, clean design with no frameworks  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6+)
